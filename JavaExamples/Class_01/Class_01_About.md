# For Loops 
### Java example
```Java
// For loop example #1
for (b : args) {
  System.out.println(b);
}
// For loop example #2
for (int i = 0; i < args.length; i++) {
  System.out.println(args[i]);
}
```
### Python example
```Python
# Just for reference, this is what that
# same loop would look like in Python
for b in args:
  print(b)
```
